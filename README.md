# Git Dumper

Git Dumper Tool.

Use asdf or pyenv to install latest python version.

Features:

- Mass Dumping.
- Parse `.gitignore`.
- Can download web accessible files.

```bash
$ pip install gitdumper-tool
$ pipx install gitdumper-tool
$ gitdumper -h
$ gitdumper url [... url]
$ gitdumper < urls.txt
$ girdumper url 2> err.log
```
